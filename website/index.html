UPX — Universal Phone eXtended

An open, modular hardware standard for smartphones — like ATX for PCs.

Read the whitepaper Join the repo Community chat

Open & OS‑agnostic

Linux first, Android‑compatible via containers. Any OS can implement the UPX HAL.

Modular by design

Compute, radios, cameras, displays, batteries, and accessories on a shared connector.

Safe by default

Hazardous modules require explicit owner consent and hardware interlocks.
